MiniBatteryLogger Help

Hidden Preferences

To set a hidden preference, open the Terminal and type the following command, then press Enter.

$ defaults write it.emeraldion.cocoa.MiniBatteryLogger PreferenceName PreferenceValue

To read the current value of a hidden preference, type the following command, then press Enter.

$ defaults read it.emeraldion.cocoa.MiniBatteryLogger PreferenceName PreferenceValue

Available Hidden Preferences

MBLFancyScaleColors

Boolean. When set to YES, it causes the labels of the scales in the chart view to have the same color of the corresponding graph.

MBLDebugMode

Boolean. Enables the debug mode. It causes a Demo Battery to be added to the list of batteries.
